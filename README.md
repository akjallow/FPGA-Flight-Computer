# FPGA-Flight-Computer
An FPGA Flight Computer being designed around the Xilinx 7000 series 484 pin BGA Package. On-board includes dual DDR3 RAM layed out with strict timing and return path requirements in a Fly-by topolgy. High-Speed interfaces such as Ethernet and USB High Speed for High Speed data Transfer. Sensor Peripherals include Lidar integrated via the RS-485 communication interface, Camera interface connected to the ZYNQ via MIPI CSI-2 , GPS,LORA and Inertial Measurement Unit (IMU) . The Primary purpose of this board is to automate landing in Unmanned Aerial Vehicles (UAVs).
